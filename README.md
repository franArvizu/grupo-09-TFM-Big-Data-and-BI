# grupo-09-TFM-Big-Data-and-BI
Repositorio académico con scripts para la obtención, limpieza e integración de datos conceptuales sobre Lean Manufacturing, ISO, SPC y Six Sigma. Incluye web scraping, uso de motores de búsqueda, preparación de datasets y análisis exploratorio como base para una plataforma gamificada de capacitación en manufactura.
